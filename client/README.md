## Folder Structure

```
- public               # bisa diakses public
- src
  - assets             # asset non-public (gambar)
  - components         # component yang dapat digunakan berulang kali
  - hooks              # react hooks
  - pages              # halaman - halaman
  - main.js            # react bootstrapper
  - App.js             # routing pages
  - firebase.js        # firebase untuk image
```
